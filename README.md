# Основной геймплей 
Игрок управляет перекрёстком (или несколькими) и:
 •	переключает светофоры (кнопки / клики)
 •	следит за потоком машин
 •	предотвращает аварии
# Ключевые механики 
1. Светофоры
 •	2 состояния зеленый/красный
 •	Игрок вручную переключает
2. Машины
 • Едут по дорогам
 • Останавливаются на красный
 • Если впереди машина-тоже останавливаются
3. Шкала агрессии
   У каждой машины есть терпение, если долго стоит,. начинает сигналить, также можно добавить проезд на красный, если "терпение" заканчивается
4. Спецтранспорт-фишка проекта
   Скорая/полиция едут быстрее, игнорируют правила, игрока должен подстроиться
# Конец игры
Авария- если две машины пересеклись
